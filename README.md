# Strange Glyph

The studio site for **Strange Glyph, LLC** — an independent software studio.

Live at [strangeglyph.com](https://strangeglyph.com).

## About

A single-page site for the studio: an animated hero, studio philosophy, an apps showcase, and contact. Built to feel calm and precise, with a custom crystal/glyph animation as the centerpiece.

## Stack

Single self-contained `index.html` — hand-written HTML, CSS, and vanilla JavaScript with no build step or framework. Hosted on GitHub Pages via a custom domain (`CNAME`).
